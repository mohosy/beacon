# Beacon

A browser SRE burn-rate simulator for analyzing SLO risk under traffic and incident conditions.

## Highlights
- 24-hour request/error stream simulation with optional incident injection.
- 5-minute and 1-hour burn-rate calculations against configurable SLO targets.
- Error-budget usage and paging-trigger estimation.
- Canvas visualizations for error-rate and burn-rate behavior.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- CSS dashboard UI

## Run
Open `index.html` in your browser.
